    Using the ThirdAI Library (Python)

    Step 1: Download a License

    To use the ThirdAI Python library, you will first need a valid license file. You may skip this step if you have already downloaded a license. To get a free trial license, please fill out this form. After submitting the form, you will be emailed a unique link to download the license.

    To download the license, simply run:

    wget your_download_link ~/license.serialized

    This will download the license file to your home directory. Please note: a valid, unexpired license must be placed in your home directory. If a valid license is not found, attempting to create a BOLT model or MaxFlash index will fail.

    You are now ready to use the ThirdAI library!

    Step 2: Install the Library

    To install the ThirdAI package, you can simply use your favorite package manager. For example, using Pip or Conda:

    pip3 install thirdai

    conda install thirdai

    The package is distributed in precompiled wheels. Currently, we support Linux 64 bit x86 and MacOS Arch64.

    Step 3: Get Started

    To get started, you can download one of the demo notebooks in our demos repo. To download the DocSearch demo, for example:

    wget https://raw.githubusercontent.com/ThirdAILabs/Demos/main/DocSearch.ipynb

    For API documentation, please visit thirdai.com/docs.



    License Agreement

    By downloading and using the ThirdAI library, you agree not to use the software for commercial purposes, not to redistribute the software, not to violate the terms of the license, and not to disassemble the code.

    If you would like to use ThirdAI software for commercial purposes, please reach out to us to discuss our enterprise license.

    Free Trial Expiration

    By default, the free trial license is valid for 60 days. However, you can always send us an email or fill out the license form again to get a link to a new license file. Running

    wget your_new_download_link ~/license.serialized

    will override the old license file. You will now be able to use the ThirdAI package again.
